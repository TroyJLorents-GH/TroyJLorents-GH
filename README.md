# About Me:
I am a full-stack engineer and Web Application Developer at Arizona State University's School of Computing and Augmented Intelligence (SCAI). With 8+ years of experience, I build robust, scalable applications and AI-powered solutions for higher education, HR, and research. I also founded AutomateFlows.io, delivering intelligent automation services for small businesses. Before ASU, I was a Software Engineer at an insurance tech company, designing solutions in fast-paced, data-driven environments.

My passions include agentic AI, automation, and bringing ideas to life with code. I love working at the intersection of AI, cloud, and web, using Python, C#, React, .NET, Azure AI Foundry, AWS, and more. Recent projects include:

- Azure AI Foundry agents with Cosmos DB, semantic search, and RAG pipelines
- Document intelligence platform with Agentic RAG, embeddings, and hybrid vector search (Azure AI Search)
- AI-powered resume matching and job advisory tools (Foundry agents + Azure AI Search)
- Student hiring and assignment systems (React, FastAPI, .NET, Azure)
- Mobile productivity app with React Native / Expo SDK 55
- Custom AI chatbots for document Q&A (OpenAI, Flask, React)

I'm always exploring new ways to make technology more useful and accessible. Whether it's building production-grade APIs, deploying AI agents, or streamlining business processes, I'm happiest when I'm solving real-world problems.
